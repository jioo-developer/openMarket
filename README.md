오픈마켓 서비스는 판매자와 구매자를 구별하여 판매자가 상품을 등록, 판매하며 구매자는 구매하는 서비스입니다.
`React` 를 사용하여 상품 등록, 결제, 상품에 대한 CRUD를 직접 구현합니다.

[요구사항](https://silver-lumber-2a3.notion.site/6327f68c03704aebb9ceede23cdbe8e1)
[구현 페이지]

1. 구매자

- 로그인 페이지
- 회원 가입 페이지
- 상품 목록 페이지
- 상품 상세 페이지
- 장바구니 페이지
- 주문/결제 페이지

2. 판매자

- 로그인 페이지
- 회원 가입 페이지
- 상품 목록 페이지
- 상품 상세 페이지
- 판매자 센터 페이지
- 상품 등록 페이지
