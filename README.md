
`API 오류로 인한 작업 중단`

오픈마켓 서비스는 판매자와 구매자를 구별하여 판매자가 상품을 등록, 판매하며 구매자는 구매하는 서비스입니다.
`React` 를 사용하여 상품 등록, 결제, 상품에 대한 CRUD를 직접 구현합니다.
